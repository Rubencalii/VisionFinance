VisionFinance 💰 Plan Completo del Proyecto 📌 Descripción General VisionFinance es una aplicación web de gestión de gastos inteligente orientada a autónomos, pequeñas empresas y usuarios domésticos. Su diferenciador principal es el escaneo OCR de tickets mediante IA, que automatiza el proceso de registro de gastos eliminando la introducción manual de datos. Objetivo del proyecto: Portfolio profesional + posible lanzamiento como producto real. 🎯 Público Objetivo Autónomos, Pequeñas empresas, Uso personal. ✨ Funcionalidades (MVP) - Autenticación: Registro e inicio de sesión seguro. - Escaneo OCR con IA: Extracción de datos de tickets (comercio, fecha, bases, IVA, total). - Categorización automática: Basada en palabras clave. - Listado y búsqueda: Filtros por comercio o fechas. - Dashboard de estadísticas: Gráficos de gastos por categoría. - Configuración API Key: Modelo "Bring Your Own Key" para Anthropic Claude. 🛠️ Stack Tecnológico React, TailwindCSS, shadcn/ui, Recharts, Node.js, PostgreSQL, Supabase, Claude API. 🏗️ Arquitectura & Modelo de Datos Sistema robusto con cifrado AES-256 para las API Keys y flujo de confirmación humana para el OCR.
